# 완장봇 1.3.0 릴리스 범위 정리

## 1.3.0에 포함
- 공통 차단/삭제 정책 구조 정비
- delete-only 처리 방식 도입
- 필터별 개별 차단 설정 구조 도입
  - keyword
  - user
  - nickname
  - url
  - voice
  - image
  - spam
  - yudong
  - kkang
  - ai
- AI 필터 연동 안정화
  - Gemini direct / Groq 호출 경로 정리
  - AI 배치 결과 파싱 안정화
  - AI BLOCK 즉시집행 복구
  - AI 개별 차단 설정 실제 집행 반영
- 스냅샷/차단 이력 연계 안정화
- 디버그 로그 정리
  - 내부 추적 로그는 디버그 모드에서만 출력

## 1.3.1로 이월
- 해외 IP 차단 기능
- 갤러리 설정 자동 갱신 기능
- 금지어 필터 특수 조건의 남은 세부 보강

## 릴리스 직전 체크
- versionName / 홈 화면 버전 표기에서 beta 문구 제거
- 사용자 노출 로그 과다 여부 최종 점검
- 릴리스 APK 빌드/업로드
- 패치노트 및 매뉴얼 반영
