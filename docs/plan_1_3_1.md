# 완장봇 1.3.1 작업 계획안

## 목표
- 1.3.0에서 이월된 기능을 사용자 흐름 기준으로 정리해 반영한다.
- 기존 1.3.0에서 정리한 공통 차단/삭제 정책 구조를 재사용한다.
- 새 기능도 기존 beta workflow(빌드 → 업로드 → 노트 → 커밋)를 유지한다.

---

## 1.3.1 우선 범위
1. 해외 IP 차단
2. 갤러리 설정 자동 갱신
3. 금지어 필터 특수 조건 보강

---

## 기능별 방향

### 1) 해외 IP 차단
목표
- ID/IP 필터 계열에서 작성자 IP가 국내 대역이 아닌 경우 차단할 수 있도록 한다.

예상 작업
- 작성자 IP 추출 경로 재확인
- 국내/해외 판별 로직 추가
- 설정 UI 추가
- 활동 로그 및 차단 이력 문구 정리

주의
- 1차는 IPv4 기준으로 시작
- 불명확한 IP 형식은 보수적으로 처리

### 2) 갤러리 설정 자동 갱신
목표
- DC 쪽 설정 변경이 생겨도 앱이 수동 재입력 없이 따라갈 수 있게 한다.

예상 작업
- 갤러리 설정 확인 요청 경로 정리
- 자동 갱신 주기/조건 정의
- 실패 시 재시도 및 로그 정리
- 설정 UI 노출 범위 검토

주의
- 과도한 요청이 되지 않도록 주기 관리 필요
- 실패 로그는 사용자에게 이해되는 수준으로 유지

### 3) 금지어 필터 특수 조건 보강
목표
- 금지어 필터를 더 정교하게 운영할 수 있도록 특수 조건을 보강한다.

예상 작업
- 현재 구현 범위 재점검
- 남은 조건 항목 확정
- UI/설정 저장/실행 경로 연결
- 활동 로그와 차단 사유 문구 정리

주의
- 규칙이 많아져도 처음 사용자 기준으로 이해 가능한 UI 유지
- 기존 금지어 필터 동작과 충돌 없도록 점검
